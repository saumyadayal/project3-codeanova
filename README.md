# 📊 ChurnShield – Customer Churn Prediction Dashboard

## 📌 Project Overview

ChurnShield is a Data Science project developed as part of the **Code A Nova Internship Program**.

The project analyzes telecom customer data to identify factors contributing to customer churn and builds machine learning models capable of predicting whether a customer is likely to leave the service.

The final goal is to present these insights through an interactive Streamlit dashboard for business users.

---

## 🎯 Objectives

- Clean and preprocess telecom customer data
- Perform Exploratory Data Analysis (EDA)
- Identify important churn factors
- Train and compare Machine Learning models
- Predict customer churn
- Deploy predictions through a Streamlit dashboard
